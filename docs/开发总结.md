# 视频混剪工具开发总结

## 实现范围

- 将模板首页替换为视频混剪操作界面，支持选择长视频、片段输出目录、合并输出目录。
- 支持设置分段秒数、视频叠加文字、文字颜色、文字位置。
- 主进程通过 `ffmpeg`/`ffprobe` 执行视频处理：分段、叠字、统一转为 1920 x 1080 的 16:9 MP4。
- 片段生成后自动生成 concat 清单，并输出一个合并后的混剪视频。
- 启动界面会检测本机是否可用 `ffmpeg` 和 `ffprobe`。

## 主要文件

- `app/app.tsx`：视频混剪工具界面与表单状态。
- `app/styles/app.css`：工具页面样式、预览区、响应式布局。
- `lib/conveyor/schemas/video-schema.ts`：视频处理 IPC 参数与返回值校验。
- `lib/conveyor/api/video-api.ts`：渲染进程调用视频能力的 API。
- `lib/conveyor/handlers/video-handler.ts`：文件选择、目录选择、FFmpeg 检测和视频处理逻辑。
- `lib/main/app.ts`：注册视频处理 IPC，并调整窗口标题和尺寸。

## 运行依赖

本功能依赖本机安装 FFmpeg，并且 `ffmpeg`、`ffprobe` 可以在系统 PATH 中直接调用。未检测到时，界面会提示需要安装 FFmpeg。

## 输出规则

- 片段文件输出到用户选择的片段目录。
- 合并文件输出到用户选择的合并目录。
- 输出视频统一为 `1920x1080`、`H.264 + AAC`、`.mp4`。
- 文字为空时只进行分段和画幅处理，不叠加文字。
